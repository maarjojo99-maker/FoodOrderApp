# Food Order App

A complete food ordering application with cart management and checkout.

## Features
- Interactive menu with product details.
- Real-time cart management.
- Integration with a backend API for fetching meals and submitting orders.
- Checkout form with validation.

## Core Concepts
- Side effects (useEffect) for data fetching.
- Context API for global state.
- Modal management for the cart and checkout.

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
